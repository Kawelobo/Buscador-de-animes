# 🎬 Buscador de Animes (CLI)

Projeto desenvolvido para a **Entrega Intermediária** do BootCamp. Esta é uma aplicação de linha de comando (CLI) construída em Python que consome uma API pública para buscar e exibir informações detalhadas sobre animes (título, episódios, nota e sinopse).

## 🎯 Funcionalidades e Requisitos Atendidos
- **Integração com API Pública:** A aplicação consome dados reais da [Jikan API](https://docs.api.jikan.moe/) (API baseada no MyAnimeList).
- **Testes de Integração:** Implementação de testes automatizados com a biblioteca `pytest` para validar e garantir o fluxo de comunicação com o serviço externo.
- **Boas Práticas de Git:** Desenvolvimento guiado por Issues, criação de Pull Request e versionamento focado na branch `entrega-intermediaria`.

## 🚀 Tecnologias Utilizadas
- **Python 3**
- **Requests** (para realizar as requisições HTTP GET)
- **Pytest** (para a suíte de testes de integração)
